🔧 Firebase Setup
One-time · takes ~5 minutes
🚨 Important: This app must be opened via http:// or https:// — NOT from a file on your computer. If you see a Google sign-in error, follow the "How to Host" steps below first.
🌐
How to Host This App (Free — 3 minutes)
Option A — GitHub Pages (Recommended):
1. Go to github.com and create a free account
2. Click New repository → name it gamepoints → set to Public → Create
3. Click Add file → Upload files → upload GamePointsManager.html
4. Rename the file to index.html after uploading
5. Go to Settings → Pages → Source: main branch → Save
6. Your app will be live at: https://yourusername.github.io/gamepoints
7. Open that URL in Chrome on your phone — done! ✅

Option B — Firebase Hosting (also free):
In Firebase Console → Hosting → Get Started → follow the steps there.
⚠️ Complete these steps once. After that, just sign in with Google on any device.
1
Create a Firebase Project
Open console.firebase.google.com → Sign in with Google → Click Add project → Name it (e.g. GamePoints) → Click through and hit Create project.
2
Register a Web App
Inside your project → Click the </> Web icon → Give it any name → Click Register app. You'll see a firebaseConfig block — keep this page open, you'll need it in Step 5.
3
Enable Google Sign-In + Add Your Domain
Left sidebar → Authentication → Get Started → Sign-in method tab → Click Google → Toggle Enable → Add your email as support email → Save.

Then click the Authorized domains tab and click Add domain. Add your GitHub Pages URL, for example:
yourusername.github.io
This is required — Google will block sign-in from any domain not listed here.
4
Create Firestore Database
Left sidebar → Firestore Database → Create database → Choose Start in test mode → Pick any region → Enable.

Then click the Rules tab, replace everything with this and click Publish:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }
5
Paste Your Config Here
From the firebaseConfig object in Step 2, copy each value below:
API Key
Auth Domain
Project ID
Storage Bucket
Messaging Sender ID
App ID
